# Settings

The Settings menu allows you to customize your launcher experience and manage your data. You can access Settings by long-pressing an empty area on the home screen and selecting **Settings**.

## Backup and Restore

Milki Launcher provides a built-in backup system to help you safeguard your home screen layout, widgets, and settings.

### Export Backup

You can export your current launcher state to a JSON backup file. This backup includes:

- Pinned apps, files, and folders
- Custom action shortcuts
- Gesture configurations

_Note: widgets restored from a backup file require manual re-binding._

### Import Backup

If you've previously exported a backup or are moving to a new device, you can import your `.json` backup file. Importing a backup will **replace** your current settings and home screen layout with the saved configuration.
