# Home Screen

The Milki Launcher home screen is your curated command center. Place anything under your fingertips—apps, app shortcuts, files, contacts, and widgets—and customize every gesture to match your workflow.

## Placing Items

Your home screen supports a variety of item types. Simply drag anything from the search dialog or app drawer onto the home screen to place it.

### App Icons

Place your most-used apps front and center. Long-press any app from the search dialog or the drawer and drag it to your desired spot.

## No Duplicates

Milki prevents duplicate items on the home screen. If you drag an existing item from one cell to another, it **moves** instead of copying. This keeps your home screen clean and intentional.

## Gesture Actions

Customize what happens when you interact with the home screen. Every gesture (trigger) can be mapped to any action.

### Available Gestures

| Gesture    | Description                         |
| ---------- | ----------------------------------- |
| One Tap    | Single tap on an empty area         |
| Double Tap | Two quick taps on an empty area     |
| Swipe Up   | Swipe upward from the bottom/middle |
| Swipe Down | Swipe downward from the top/middle  |

More gestures are planned for future releases.

### Available Actions

Each gesture can be mapped to one of the following actions:

| Action             | What It Does                                    |
| ------------------ | ----------------------------------------------- |
| Show Notifications | Expands the notification panel                  |
| Open Drawer        | Opens the app drawer                            |
| Open App           | Launches a specific app of your choice          |
| Open App Shortcut  | Launches a specific app shortcut of your choice |
| Lock Screen        | Locks the device                                |
| Do Nothing         | Disables the gesture                            |

## Interacting with the Home Screen

### Home Button

Tapping the home button instantly opens the Search Dialog, putting your command center one tap away.

### Context Menu

Long-press on any empty area of the home screen to open the context menu. From here you can:

- **Open Settings**
- **Open Widget Picker** to add a new widget

### App Shortcuts

Long-press any app icon to open its context menu, which displays available app shortcuts provided by the app. You can drag any of these shortcuts and drop them onto the home screen to create a dedicated icon for quick access.
